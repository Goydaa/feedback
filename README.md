Feedback Management System 🚀
Информационная система для сбора и управления пользовательскими отзывами, разработанная в рамках практической подготовки в ММКЦТ «Академия TOP».

📋 Описание проекта
Данный сервис позволяет пользователям оставлять отзывы через веб-интерфейс, а администратору — просматривать и модерировать их в защищенной панели управления. Проект демонстрирует реализацию полноценного CRUD-приложения с разделением прав доступа.

🛠 Стек технологий
Backend: Python, Flask.

Database: SQLite, SQLAlchemy (ORM).

Auth: Flask-Login (защита маршрута /admin).

Frontend: HTML5, CSS3, Jinja2.

Deployment: Render (PaaS).

⚙️ Основные функции
Публичная форма: Валидация полей и отправка отзывов в базу данных.

Безопасность: Доступ к панели управления защищен авторизацией (admin/admin).

Стабильность: Реализован паттерн PRG (Post/Redirect/Get) для предотвращения дублирования записей.

CI/CD: Автоматический деплой из репозитория GitHub на сервер Render.

🚀 Как запустить локально
Клонируйте репозиторий.

Установите зависимости: pip install -r requirements.txt.

Запустите приложение: python app.py.

Откройте в браузере: [http://127.0.0.1:5000](http://127.0.0.1:5000)
