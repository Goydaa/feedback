### 📝 Описание проекта
Информационная система для сбора и управления пользовательскими отзывами. Проект разработан в рамках учебной практики в **ММКЦТ «Академия TOP»**. Сервис позволяет пользователям оставлять обратную связь, а администраторам — безопасно управлять данными через защищенную панель.

---

## 🛠 Стек технологий
Проект построен на базе современного стека для веб-разработки:

*   **Backend:** `Python 3.10+` & `Flask`
*   **ORM:** `SQLAlchemy` (взаимодействие с БД)
*   **Database:** `SQLite` (надежное локальное хранилище)
*   **Security:** `Flask-Login` (сессионная авторизация)
*   **Deployment:** `Render` + `Gunicorn`

---

## 🚀 Основные возможности

### 👤 Для пользователя:
*   **Форма обратной связи:** Удобный интерфейс для написания отзывов.
*   **Валидация:** Проверка полей на стороне сервера (защита от пустых записей).
*   **Стабильность:** Использование паттерна **Post/Redirect/Get** для предотвращения дубликатов при обновлении страницы.

### 🔐 Для администратора:
*   **Защищенный вход:** Доступ к разделу `/admin` закрыт декоратором `@login_required`.
*   **Панель управления:** Просмотр всех накопленных данных в виде структурированной таблицы.
*   **Сессии:** Возможность безопасного выхода из системы (`/logout`).
